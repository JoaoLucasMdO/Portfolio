Sobre Mim

Atualmente, sou aluno do curso de Desenvolvimento de Software Multiplataforma na Fatec Votorantim, onde tenho aprimorado minhas habilidades em desenvolvimento web. Já participei de diversos projetos, nos quais criei sistemas que consomem APIs REST que também desenvolvi. Essas APIs utilizam bancos de dados como MongoDB e Firebase. No desenvolvimento do Back-End, trabalho com linguagens como JavaScript e TypeScript, enquanto no Front-End, utilizo frameworks como Next.js ou HTML puro. Em todos os projetos, busco sempre oferecer a melhor experiência ao usuário, garantindo uma visualização eficiente e intuitiva dos dados.
